# Experiment 3.3.1 - Todo Application with CRUD Operations

## Aim
To build a full-stack todo application with basic CRUD operations using the MERN stack.

## Features
- Add new todo items
- View all todos
- Update todo text and completion status
- Delete todos

## Run

### Backend
```bash
cd todo-app/backend
npm install
npm start
```

### Frontend
```bash
cd todo-app/frontend
npm install
npm run dev
```

## Default URLs
- Backend: `http://localhost:5000`
- Frontend: `http://localhost:5173`
